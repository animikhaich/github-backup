# Security Policy

## Reporting a Vulnerability

We take the security of our software seriously. If you suspect you have found a vulnerability, we want to know so we can fix it as quickly as possible.

**Please do not open public issues for security vulnerabilities.**

To report a vulnerability, please email `animikhaich@gmail.com`.

We will:

* Acknowledge your report as soon as possible.
* Verify the vulnerability and determine its impact.
* work on a fix and release it.
* Credit you for the discovery (if you wish).

Thank you for helping keep our community safe!
